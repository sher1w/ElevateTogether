
Interactive Habit-Tracking Community
This project is a habit-tracking platform that focuses on both personal progress and community motivation.
Users can create habits, track their daily performance, join groups, and participate in challenges to stay consistent.

Features:

1. Habit Creation & Tracking
Users can create custom habits (e.g., reading, workouts, meditation).
Daily check-ins with streak tracking.
Optional reminders or notifications.
Habit-wise stats like success rate and missed days.

2. Community Challenges
Users can join public challenges or create private ones with friends.
Small group interactions for motivation.
Challenge updates can be shared within the group.

3. Gamification
Badges based on consistency and streak length.
Levels or XP for completing habits regularly.
Leaderboards for friendly competition.

4. Data Visualization
Simple charts showing habit progress over time.
Weekly and monthly trend views.
Overall dashboard summarizing all active habits.

Tech Stack:
Frontend: React 
Backend: Node.js 
Database:  Firebase
Charts: Chart.js
